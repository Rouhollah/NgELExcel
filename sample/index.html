Sample Data 1

[{ id: 10, name: 'ehsan' }, { id: 11, name: 'moslem' }, { id: 12, name: 'iman' }, { id: 13, name: 'javad' }]

Sample Data 2

[{ id: 20, name: 'maryam' }, { id: 21, name: 'elham' }, { id: 22, name: 'sahar' }, { id: 23, name: 'sara' }]

Example #1 :

Quick conversion to Excel
NgELExcel.Excel(SampleData1);

Example #2 :

You can customize your Excel output by rendering function
NgELExcel.Excel(SampleData1,[ { title: "rows", name: "#rowNumber" }, { title: "Identity", name: "id" }, { title: "UserName", name: "name" } ]);

Example #3 :

Limit the output records to the third argument of this service
NgELExcel.Excel(SampleData1,null,2);

Example #4 :

You can have multiple outputs in an Excel file
NgELExcel.Excel([SampleData1,SampleData2]);

Example #5 :

Finally, a combination of previous methods can be implemented together
NgELExcel.Excel([SampleData1, SampleData2],[null,null],[2,3]);
